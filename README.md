# vuepress-theme-qblog

## Install

```
npm i vuepress-theme-qblog -D
# OR yarn add vuepress-theme-qblog -D
```

## Usage

```
// .vuepress/config.js
module.exports = {
  theme: 'vuepress-theme-qblog',
  themeConfig: {
    // Please head documentation to see the available options.
  }
}
```
