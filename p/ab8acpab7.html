在写博客首页 哔哔 组件的时候首先想到的就是 Swiper 这个库也是用了好久了，现在都更新到 Swiper8 了，查看官方的文档后给出了以下示例代码。

<template>
  <swiper
      :slides-per-view="3"
      :space-between="50"
      @swiper="onSwiper"
      @slideChange="onSlideChange"
  >
    <swiper-slide>Slide 1</swiper-slide>
    <swiper-slide>Slide 2</swiper-slide>
    <swiper-slide>Slide 3</swiper-slide>
    ...
  </swiper>
</template>
<script>
// Import Swiper Vue.js components
import {Swiper, SwiperSlide} from 'swiper/vue';

// Import Swiper styles
import 'swiper/css';

export default {
  components: {
    Swiper,
    SwiperSlide,
  },
  setup() {
    const onSwiper = (swiper) => {
      console.log(swiper);
    };
    const onSlideChange = () => {
      console.log('slide change');
    };
    return {
      onSwiper,
      onSlideChange,
    };
  },
};
</script>

引入到项目后发现有报错，遂 Google 了一圈发现 Swiper 的最新几个版本都是对 Vue3 的支持，Vue2 是不能使用该写法的，那既然不能用组件咱们还是老方法来直接使用 Swiper 暴漏出的类直接实例化。

<template>
  <div ref="swiper" class="swiper">
    <div class="swiper-wrapper">
      <div class="swiper-slide" v-for="(el,i) in bbLIst" :key="i">{{ el.title }}</div>
    </div>
  </div>
</template>

<script>
import Swiper, {Autoplay} from 'swiper'
import 'swiper/swiper-bundle.min.css'

export default {
  mounted() {
    this.initSwiper()
  },
  methods: {
    initSwiper() {
      new Swiper(this.$refs.swiper, {
        modules: [Autoplay],
        loop: true,
        direction: 'vertical',
        autoplay: {
          delay: 2000,
          disableOnInteraction: false,
        },
        speed: 800,
      })
    }
  }
}
</script>

注意 div 标签上的类名不可以漏掉，然后因为 哔哔 是上下滚动嘛，就必须得禁用用户手动触发轮播了，手动给最外层的 div 添加一个 swiper-no-swiping 类名即可
